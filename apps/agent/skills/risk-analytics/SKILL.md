# UC3 Risk Analytics Visualization Skill

Use English for user-facing text. Always use the real Foundry/Genie result; never invent numbers.

Visual workflow:
1. Acknowledge the business question briefly.
2. Show a visualization plan.
3. Query Foundry/Genie.
4. Render controlled components only.
5. Explain the insight and suggest a drill-down.

Mapping:
- Exposure by country: KpiStrip + BarChartCard.
- Quarter comparison: ComparisonChart.
- Brokers by claims: BarChartCard + InsightTable.
- Overdue balance by risk class: BarChartCard.
